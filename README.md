# AmazonLogo

任意の英単語でAmazon風ロゴを生成するWebツールです。

## デモ

https://ogawakeito.github.io/AmazonLogo/

## 使い方

1. テキストボックスに英単語（a〜z）を入力する
2. 入力した文字列が自動的にAmazon風ロゴとして表示される

## 技術スタック

- HTML5
- CSS3
- Vanilla JavaScript

## ファイル構成

```
AmazonLogo/
├── index.html   # メインページ
├── make.js      # ロゴ生成処理
└── font/        # ロゴ用画像素材
    ├── a.gif〜z.gif  # アルファベット画像（各文字）
    └── arrow.png     # Amazon風の矢印画像
```
